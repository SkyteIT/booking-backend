# Quick EF Migration Commands Reference

## Essential Commands:

# 1. Check status
dotnet ef migrations list --project src\Ube.Infrastructure --startup-project src\Ube.Api

# 2. Create migration
dotnet ef migrations add AddNewFeature --project src\Ube.Infrastructure --startup-project src\Ube.Api

# 3. Apply to database
dotnet ef database update --project src\Ube.Infrastructure --startup-project src\Ube.Api

# 4. Generate SQL script
dotnet ef migrations script --project src\Ube.Infrastructure --startup-project src\Ube.Api

## Troubleshooting:

# Check for pending changes
dotnet ef migrations has-pending-model-changes --project src\Ube.Infrastructure --startup-project src\Ube.Api

# Remove last migration (if not applied)
dotnet ef migrations remove --project src\Ube.Infrastructure --startup-project src\Ube.Api

# Check database info
dotnet ef dbcontext info --project src\Ube.Infrastructure --startup-project src\Ube.Api

## Advanced:

# Update to specific migration
dotnet ef database update MigrationName --project src\Ube.Infrastructure --startup-project src\Ube.Api

# Script between migrations
dotnet ef migrations script FromMigration ToMigration --project src\Ube.Infrastructure --startup-project src\Ube.Api</content>
